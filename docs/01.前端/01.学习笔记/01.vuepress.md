# VuePress

## 项目搭建

### 1. 创建并进入新目录

``` sh
mkdir vue-press-demo
cd vue-press-demo
```

### 2. 初始化

``` sh
npm init -y
```

### 3. 安装 VuePress

``` sh
npm install -D vuepress
```

### 4. 配置启动/打包脚本，在 package.json 添加一些 script

``` json
{
  "scripts": {
    "dev": "vuepress dev docs",
    "bulid": "vuepress build docs",
  }
}
```
